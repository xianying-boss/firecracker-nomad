# Step 2: Firecracker Installation

This step installs the Firecracker binary.

## What this step does:
- Downloads the Firecracker release from GitHub.
- Extracts and installs the binary to `/usr/local/bin/firecracker`.
- Verifies the installation by checking the version.

## How to run:
```bash
sudo bash run.sh
```

## Default Version:
- Firecracker v1.7.0
